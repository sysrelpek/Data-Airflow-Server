# Data Airflow Server

A modern, extensible data orchestration platform built on Apache Airflow with a clean hexagonal architecture.

## Features

- Dynamic DAG generation from YAML/JSON manifests
- Pluggable storage adapters (Postgres, File, etc.)
- Clean separation between business logic and infrastructure
- Designed for easy extension and testing

## Getting Started

```bash
git clone https://github.com/sysrelpek/Data-Airflow-Server.git
cd Data-Airflow-Server
```
